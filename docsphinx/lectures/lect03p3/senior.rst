The problem
===========

Lemma
-----

Lemma: In radix R, digit root of number N, :math:`D_R(N) = N mod (R-1)`.

Proof:
For a 2-digits R radix number n, :math:`n = R\cdot a + b`, where :math:`a, b \in {0, 1, ..., R-1}`.

.. math::

    n mod (R-1) = (R \cdot a + b) mode (R-1)
..
