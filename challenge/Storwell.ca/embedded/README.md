# About

Answer to Storwell challenges.

# Compile & run

Tested on Ubuntu 16.0.4 LTS, gcc 5.4.0.

- q1

```commandline
    g++ g1.c
    ./a.out 12 12
    12x12
```

- q2

```commandline
    g++ g2.c
    ./a.out 15840
    cGp
```

Or test with the Python script for verifying all the inputs:

```commandline
    g++ g2.c
    pytyon3 test.py
    15840   cGp
    ...
```
