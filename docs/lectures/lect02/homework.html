Homework¶

Task 1 Exercise¶

1. If a logic function has three inputs, how many rows must the truth table have to contain all possible states? Justify your answer.

  1. For the following functions, construct a truth table and draw a circuit diagram.

    (a). \(y(A,B) = \overline{AB}+\overline{B}\)

    (b). \(y(A,B,C) = \overline{AC} + BC\)

    (c). \(y(A,B,C) = (A ⨁ B)\overline{C}\)

    (d). \(y(A,B,C) = \overline{\overline{A+B}\space\overline{B+C}}\)

    (e). \(y(A,B,C,D) = \overline{A}B\overline{C} + (A ⨁ B)C + \overline{A}\overline{B}C\overline{D} + ABC\)

3. Design a 4-input NAND gate using two 2-input NAND gates and one 2-input NOR gate. Hint: Use DeMorgan’s law.

  1. Simplify

\(\overline{ABC}+ \overline{AB}C+ \overline{A}B\overline{C}+ A\overline{BC}+ A\overline{B}C\)

\(\overline{\overline{X\overline{Y} + XYZ} + X (Y + X\overline{Y}) }\)

\((A + B)(\overline{A} + C)(B + C)\)

\(XY + \overline{XZ} + X \overline{Y} Z (XY + Z)\)

Answer: \((\overline{A}+\overline{B}+\overline{C}), \quad0, \quad AC+B\overline{A}, \quad1\)

Task 2 Prefix Expression *¶

Attention

Try this after the lecture of recursive functions.

Let’s rewrite the logic in the following notations:

\[\begin{split}\begin{array}{c|c} \hline logic & operator\\ \hline NOT & ! AND & & OR & | XOR & ^\end{split}\]

Write the first four expression in step 4, task1 in prefix form. E.g (A OR B) AND X’s prefix form is:

& | A B X.

Then implement a program to evaluate all answer of question 4 in task 1.

Hint: the important data structure for the program is Stack.
