For Your Information:

An other string operation to replace some characters is typically like:

chars = list("String") # char = ['S', 't', 'r', 'i', 'n', 'g']
chars[3] = 'o'
s = ''.join(chars)     # s = 'Strong'

And a little bit faster:

s = "String"
s = s[:3] + 'o' + s[4:] # s = 'Strong'
